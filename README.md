# 🏥 ENARM Pro

Plataforma de preparación ENARM basada en estrategia, palabras pivote y joyas.

## 🚀 Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Vercel** (Deployment)

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/enarm-pro.git
cd enarm-pro

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Build para producción
npm run build
npm start
